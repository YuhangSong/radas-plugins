# radas-plugins

Claude Code plugins for [radas](https://github.com/YuhangSong/radas).

| Plugin | Description |
|--------|-------------|
| `radas` | Run large-scale experiments with Ray Tune and analyze results with pandas/seaborn. |
| `ask-user` | Interact with the user via email when away from the terminal. |

## Install

```
/plugin marketplace add YuhangSong/radas-plugins
/plugin install radas@radas-plugins
/plugin install ask-user@radas-plugins
```

## Update

```
/plugin update radas@radas-plugins
/plugin update ask-user@radas-plugins
```

Or enable auto-update: `/plugin` → **Marketplaces** → select `radas-plugins` → **Enable auto-update**.

## Uninstall

```
/plugin uninstall radas
/plugin uninstall ask-user
```
